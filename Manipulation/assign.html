Adding and changing HTML elements using DOM Manipulation

Pretty much anything on an HTML page can be manipulated using the Document Object Model.
For example, if I wanted to create a button on my page I could use the document.createElement function to create the button element and then the appendChild element to append it to the end of the contents of another element. Note that after I create an element I can add properties like an onclick event or an id or I can add to the contents of that element with appendChild.
